# Raheja Lunaris Website

This repository/folder contains the static website files for **Raheja Lunaris**, a luxurious residential project by Raheja Universal located in Juinagar, Mumbai.

## Project Structure

- `index.html`: Main landing page of the website.
- `privacy-policy.html`: Privacy policy page.
- `terms-conditions.html`: Terms and conditions page.
- `cookies-policy.html`: Cookies policy page.
- `thank-you.html` & `thank-you.php`: Post-inquiry thank you pages.
- `assets/`: Contains all the CSS, JavaScript, fonts, and image assets.
- `brochure.pdf`: Downloadable project brochure.
- `pdf.pdf`: MahaRERA Agent Registration Certificate.

## Hosting & Deployment

To view the website locally, you can open `index.html` in any modern web browser or serve it using a local static file server.
